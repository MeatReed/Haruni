## Install Setup

```bash
# clone git
$ git clone https://github.com/MeatReed/LavaJs-Bot.git

$ cd LavaJs-Bot

# install packages
$ yarn install # npm install

# mode dev
$ yarn dev
# mode prod
$ yarn start
```

## .env file
```
DISCORD_BOT_TOKEN=
DISCORD_BOT_PREFIX=h!
DISCORD_BOT_OWNER=
LAVACORD_HOST=
LAVACORD_PORT=

```
